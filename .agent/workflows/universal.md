### Universal Workflow Phase
1. **Initialize**: Determine scope and constraints.
2. **Plan**: Define implementation steps using `plan-writing`.
3. **Execute**: Implement changes modularly.
4. **Validate**: Verify functionality and Linting.
5. **Persist**: Update documentation.
6. **Continuous Improvement**: Apply Kaizen.
